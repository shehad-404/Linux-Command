# a) Basic Command Lines

**1: Clear all commands in Console**

```bash
clear
```
**2: Returns user name**

```bash
whoami
```
**3: returns host name**

```bash
hostname
```
**4: Shows current working directory**

```bash
pwd
```
**5: List directory contents**

```bash
ls
```
